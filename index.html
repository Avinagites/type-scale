How to Use

I like to enter the base font size of my body content. Then I select from the surrounding values for headers and small copy.

There are no rules. Just experiment and have fun. Try using the values for line-height, margins or whatever, and see what works.

Background

Most people are familiar with Tim Brown’s Modular Scale calculator. The site has raised awareness about type scale and helped improve typography on the web. It’s also the inspiration for this project.

The Modular Scale site has been a great reference over the years, but sometimes it’s difficult to choose the proper scale without a visual reference. I hope to bridge that gap with the Type Scale Calculator.
